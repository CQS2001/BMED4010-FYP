The python file RandomSampling.py aimes to randomly select 1000 samples from the fitness databank, the data were preliminary stratified into 10 groups in ascending order.
The python file MutantGenerator.py aimes to create the mutatnt file for the sampled file in EvoEF format, each mutant file will create 1000 mutants.
The MATLAB file EvoEF_Analysis.m aimes to calculate the Pearson and Spearman correlation coefficient between EvoEF ddG and EvMutation fitness.
