This section contains the files of ddG_Stability investigation for Cas9 proteins
The file EvoDDG_Cas9.xlsx contains the SaCas9 Fitness Data, SpCas9 Fitness Data and their corresponding EvoEF ddG_Stability
The file EvoDDG_Cas9_Analysis.m is the MATLAB file which was used for correlation test
The file Sa_mutantgenerator.py and Sp_mutantgenerator.py (in their sub-directory) are the Python file which were used to generate the EvoEF mutant file from those two Empirical fitness data csv file
